# ByteFlow 2.0 — Faz C1 (Persona Katmanı) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax.

**Goal:** @byteflowlabs'e isimli "senior AI engineer" personası — **Kai** — ver: video outro imzası + caption attribution. Ses YOK (edge-tts, A/B sinyaline ertelendi).

**Architecture:** İki nokta: (1) `render/src/scenes/explainer.tsx` outro'suna "— Kai · @byteflowlabs" imza satırı; (2) Gemini caption'ı follow sign-off'undan önce "Written by Kai." satırı içersin (prompt) + 8 seed caption'ı da tutarlılık için güncellensin. Sessiz-sinematik mimari korunur.

**Tech Stack:** Node 22 ESM, Motion Canvas 3.17.2, Gemini structured output, `node --test`.

## Global Constraints

- ESM only; testler `node --test`. Render `render/` içinde TSX (vite).
- Render deterministik (no Math.random/Date.now).
- Persona adı: **Kai**. Outro imzası birebir: `— Kai · @byteflowlabs`. Caption persona satırı birebir: `Written by Kai.` (follow sign-off'undan HEMEN önce, ayrı satır).
- Caption follow sign-off DEĞİŞMEZ: `Follow @byteflowlabs for AI systems, no hype.`
- Anti-injection çiti + pillar/hook/takeaway/code (Faz A/B) davranışı korunur.
- Geriye dönük: mevcut render akışı (hook→scenes→takeaway) bozulmaz; sadece outro'ya imza eklenir.
- Ses (edge-tts) bu fazın DIŞINDA.
- Commit sık; her task sonunda testler yeşil.

## Task 1: Render outro — Kai imzası

**Files:**

**Interfaces:**
- Outro artık takeaway + `— Kai · @byteflowlabs` imzası + mevcut tagline gösterir.

- [ ] **Step 1: Read the current outro block**

`explainer.tsx` içindeki `// ---- Takeaway + follow (outro) ----` bloğunu bul (Faz A'da eklendi). Şu anda `take` (TAKEAWAY, büyük) + `follow` ("follow @byteflowlabs · AI systems, no hype") gösteriyor.

- [ ] **Step 2: Add the Kai signature line to the outro**

Outro bloğunu, takeaway'in altına persona imzası + tagline gelecek şekilde değiştir. `take` (TAKEAWAY) korunur; `follow` satırının yerine iki satır:

```tsx
  // ---- Takeaway + persona sign-off (outro) ----
  yield* title().opacity(0, 0.3);
  const take = createRef<Txt>();
  const sign = createRef<Txt>();
  const tag = createRef<Txt>();
  view.add(<Txt ref={take} text={TAKEAWAY} fill={COLORS.text} fontFamily={MONO}
    fontSize={60} fontWeight={800} letterSpacing={1} opacity={0} y={-70}
    width={960} textAlign="center" textWrap />);
  view.add(<Txt ref={sign} text="— Kai · @byteflowlabs" fill={ACCENT} fontFamily={MONO}
    fontSize={38} fontWeight={600} letterSpacing={2} opacity={0} y={140} />);
  view.add(<Txt ref={tag} text="AI systems, no hype" fill={COLORS.muted} fontFamily={MONO}
    fontSize={30} letterSpacing={3} opacity={0} y={210} />);
  yield* all(take().opacity(1, 0.5), sign().opacity(1, 0.5));
  yield* tag().opacity(1, 0.4);
  yield* waitFor(1.4);
```

Not: `ACCENT`, `COLORS`, `MONO`, `TAKEAWAY`, `title` referansları dosyada zaten mevcut (Faz A). Eğer outro'da `title().opacity(0,...)` satırı yoksa (kod-sahne yolunda title farklı olabilir), mevcut yapıyı koru — sadece `follow` satırını `sign`+`tag` ile değiştir.

- [ ] **Step 3: Update the caption rule in the prompt**

`brain/generate-spec.mjs` `PROMPT` içindeki caption kuralında, follow sign-off satırından hemen önce persona satırını ekle. Mevcut caption kuralı (Faz A):

```
- caption: 3 to 5 short lines. Line 1 = the sharp claim (echo the hook). Then the insight, skimmable.
  Then a save/share CTA (e.g. "Save this before your next AI build" or "Tag someone shipping agents").
  End with EXACTLY: "Follow @byteflowlabs for AI systems, no hype."
```

Şununla değiştir:

```
- caption: 3 to 5 short lines. Line 1 = the sharp claim (echo the hook). Then the insight, skimmable.
  Then a save/share CTA (e.g. "Save this before your next AI build" or "Tag someone shipping agents").
  Then a persona line EXACTLY: "Written by Kai."
  Then the final line EXACTLY: "Follow @byteflowlabs for AI systems, no hype."
```

- [ ] **Step 4: Run tests to verify pass**

Run: `node --test brain/generate-spec.test.mjs`
Expected: PASS (yeni test + mevcut Faz A/B testleri).
