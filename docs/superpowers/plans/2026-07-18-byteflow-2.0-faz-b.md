# ByteFlow 2.0 — Faz B (Gerçek Kod Estetiği) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** ByteFlow videolarına gerçek syntax-highlight kod sahnesi ekle — soyut diyagram tek görsel dil olmaktan çıksın, "insider/otantik" kod estetiği devreye girsin. Video kurgusu: `HOOK → (CODE ve/veya DIAGRAM) → TAKEAWAY`.

**Architecture:** Motion Canvas'ın native `Code` component'i (`@motion-canvas/2d` 3.17.2, `LezerHighlighter` + `@lezer/python` parser) kullanılır — sıfırdan syntax highlighter yazılmaz. Spec'e opsiyonel `kind` alanlı yeni sahne tipi (`code`) eklenir; mevcut diyagram sahneleri `kind` yoksa eskisi gibi davranır (geriye dönük uyumlu). Gemini beyni bazı konular için illustratif (çalışır olmak zorunda olmayan) Python-vari snippet üretir. Determinizm + CI-safe (yeni sistem bağımlılığı yok — parser vite ile bundle'lanır).

**Tech Stack:** Node 22 ESM, Motion Canvas 3.17.2 (`Code`, `LezerHighlighter`, `CodeSignal`), `@lezer/python`, Gemini 2.5-flash structured output, ajv, `node --test`.

## Global Constraints

- ESM only; beyin/pipeline testleri `*.test.mjs` + `node --test`. Render `render/` içinde TSX (vite).
- `@motion-canvas/*` yalnız `render/` içinde import edilir; beyin/pipeline saf JS kalır.
- Render deterministik: render sahnesinde `Math.random`/`Date.now` YOK.
- Gemini'ye giden trend başlıkları UNTRUSTED DATA — anti-injection çiti korunur.
- **Geriye dönük uyumluluk KESİN:** `kind` alanı OPSİYONEL. `kind` yoksa veya `kind: "diagram"` ise sahne bugünkü nodes-flow diyagramı gibi render edilir. Faz A'da üretilen/mevcut seed spec'leri değişmeden çalışmalı.
- Kod doğruluğu **illustratif** — snippet gerçeğe yakın/idiomatik görünür ama çalışır/lint-geçer olmak zorunda değil (AST/lint gate YOK, bu Faz B kapsamı dışı).
- CI gotcha'ları (xvfb + `RENDER_HEADED=1` + `process.exit(0)` + dosya-stabilizasyon) DOKUNULMAZ.
- Yeni npm bağımlılığı yalnız `render/` altında (`@lezer/python`); kök `package.json` değişmez.
- Caption imzası ve pillar/hook/takeaway davranışı (Faz A) korunur.
- Commit sık; her task sonunda testler yeşil.

- [ ] **Step 3: Add the `renderCodeScene` generator**

Dosyanın sonuna (default export'un DIŞINA, ama import'lardan sonra) ekle. `ctx` = `{accent, colors, pacing}` (mevcut yapı). Süre `pacing`'e saygılı:

```tsx
function* renderCodeScene(view: any, scene: any, ctx: any) {
  const container = createRef<Layout>();
  view.add(<Layout ref={container} opacity={0} />);

  if (scene.heading) {
    container().add(<Txt text={scene.heading} fill={ctx.accent} fontFamily={MONO}
      fontSize={48} fontWeight={600} letterSpacing={1} y={-620} opacity={0.95} />);
  }

  // Kod kartı (arka plan) + Code node.
  container().add(<Rect width={980} height={880} radius={28} fill={COLORS.card}
    stroke={COLORS.stroke} lineWidth={3} shadowColor={'#00000066'} shadowBlur={40} shadowOffsetY={14} y={-10} />);

  const code = createRef<Code>();
  container().add(
    <Code ref={code} highlighter={byteflowHighlighter as unknown as LezerHighlighter}
      fontFamily={MONO} fontSize={40} offsetX={-1} x={-440} y={-360} code={''} />,
  );

  if (scene.annotation) {
    container().add(<Txt text={scene.annotation} fill={COLORS.muted} fontFamily={MONO}
      fontSize={40} fontWeight={500} y={520} width={960} textAlign="center" textWrap opacity={0.9} />);
  }

  yield* container().opacity(1, 0.4);

  // Reveal: typing (varsayılan) satır satır süreyle yazar; instant tek seferde.
  const full = scene.code as string;
  if (scene.reveal === 'instant') {
    code().code(full);
    yield* waitFor(ctx.pacing.finalDwell + 1.2);
  } else {
    // Kod tween — CodeSignal ile hedef koda "yazılır" (typing hissi).
    yield* code().code(full, Math.min(2.4, Math.max(1.2, full.length / 90)));
    yield* waitFor(ctx.pacing.finalDwell + 0.8);
  }

  yield* container().opacity(0, 0.4);
  container().remove();
}
```

Not: `code().code(target, duration)` MC 3.17.2 CodeSignal tween'idir (boş koddan hedefe = typing/diff animasyonu). `highlighter` prop cast'i tip köprüsü içindir (LezerHighlighter uyumlu).
